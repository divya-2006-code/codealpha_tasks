## 📖 Project Description
This is a responsive Image Gallery web application built using HTML, CSS, and JavaScript.  
It allows users to view, filter, and interact with images in an attractive layout.

---

##  Features
-  Image gallery with grid layout
-  Lightbox view on image click
-  Next / Previous navigation
-  Zoom functionality (scroll to zoom)
-  Category-based filtering (Nature, City, Bird, Food, Baby)
-  Modern UI with hover effects and animations
-  Fully responsive design
-  Deployed using GitHub Pages

---

##  Technologies Used
- HTML5
- CSS3
- JavaScript (Vanilla JS)
- GitHub Pages (Deployment)

---

##  Conclusion
This project demonstrates frontend development skills including layout design, interactivity, DOM manipulation, and deployment.
