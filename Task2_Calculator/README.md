##  Project Description

Smart Calculator Pro is a frontend web application developed using HTML, CSS, and JavaScript. It performs basic arithmetic operations with an enhanced user experience through a modern interface, keyboard support, and additional smart features like history tracking and theme switching.

---

##  Features

###  Core Features
- Addition, Subtraction, Multiplication, Division
- Interactive buttons and display screen
- Clear (C) and Delete (⌫) functionality
- Error handling for invalid inputs

###  Advanced Features
-  Keyboard support
-  Dark/Light theme toggle
-  Calculation history panel
-  Persistent history using localStorage
-  Responsive design
