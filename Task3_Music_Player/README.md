# 🎧 Music Player Web Application

##  Project Overview

This project is a simple and interactive **Music Player** developed using **HTML, CSS, and JavaScript**. It allows users to play audio files with essential controls and a clean user interface.

---
##  Live Demo

https://divya-2006-code.github.io/Music-Player---Task-3/

---

##  Objective

* To build a web-based music player
* To implement audio controls using JavaScript
* To design an attractive and responsive interface
* To enhance user experience with playlist and visuals

---

##  Features

*  Play / ⏸ Pause toggle
*  Next / ⏮ Previous controls
*  Progress bar with seek functionality
*  Volume control
*  Playlist with clickable songs
*  Autoplay next song
*  Album image display
*  Highlight current playing song

---

##  Technologies Used

* HTML
* CSS
* JavaScript

---

##  Working Principle

The application uses the HTML `<audio>` element for playing music. JavaScript controls the playback operations such as play, pause, and switching songs. The progress bar updates dynamically based on playback time, and user interactions are handled using event listeners.

---

##  Output

The output is a functional music player where users can:

* Play and control songs
* Navigate between tracks
* Adjust volume
* View song progress
* See album images

---

##  Conclusion

This project successfully demonstrates how JavaScript can be used to control multimedia elements. It provides an interactive and user-friendly experience while meeting all the required functionalities of a music player.
